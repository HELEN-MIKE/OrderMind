# OrderMind 下一阶段实施计划 2026-07-03

## 目标

把当前 Electron 桌面壳推进到更接近客户试用版的状态：打开后更像独立桌面应用，审单结果可以导出报告。

## 范围

本阶段只做两个可验收闭环：

1. 桌面产品细节：应用图标、窗口标题、菜单、打包资源配置。
2. 结果报告导出：结果页提供本地可下载 HTML 报告，后续再扩展 PDF/Excel。

## 不做

- 不重构为 React/Vue/Tauri。
- 不引入数据库。
- 不做 Windows 本机构建；Windows 安装包后续在 Windows 或 CI 环境生成。
- 不提交、不推送，保留给后续统一提交。

## 实施顺序

1. 为报告导出写失败测试。
2. 增加 HTML 报告渲染函数和结果页导出表单。
3. 增加 `/export-report` 本地 POST 路由，返回附件下载。
4. 为 Electron 增加窗口菜单、图标路径和打包图标资源。
5. 更新发布检查和验收记录。
6. 运行单元测试、编译检查、发布检查、CLI 样例、Mac 打包和桌面健康检查。
